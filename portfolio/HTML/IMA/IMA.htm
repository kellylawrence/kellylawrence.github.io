Interactive Media Arts Mockup

Programs Used: Adobe Illustrator CS4 and Adobe Photoshop CS4

This is a mockup for the new Interactive Media Arts website at ACC. I created this mockup for a final project in my Visual Design 4 class during my second year at ACC.

I wanted to create a professional looking layout for the site, as well as a way to show off what life is like in the course, using a large slideshow for the header. The large, accessible navigation was my way of creating a simple way for first time users to get directly to the main content they are looking for.
